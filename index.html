Datenbank erweitern

Einmalig ausführen

Da neue Spalten hinzugekommen sind, muss die Datenbank einmalig erweitert werden.
Dieses SQL im Supabase SQL Editor ausführen:

alter table kunden add column if not exists strasse text, add column if not exists konditionsgruppe text, add column if not exists listungsnummer text, add column if not exists zahlungsbedingung text, add column if not exists datum_letzte_lieferung date, add column if not exists datum_letzter_kauf date; alter table ansprechpartner add column if not exists anrede text, add column if not exists vorname text, add column if not exists zentrale text, add column if not exists email_zentrale text; alter table umsatz_verlauf add column if not exists umsatz_vj numeric(12,2), add column if not exists ae_aj numeric(12,2), add column if not exists ae_vj numeric(12,2), add column if not exists ae_abweichung numeric(12,2), add column if not exists ae_saison_fs_aj numeric(12,2), add column if not exists ae_saison_fs_vj numeric(12,2), add column if not exists ae_saison_hw_aj numeric(12,2), add column if not exists ae_saison_hw_vj numeric(12,2), add column if not exists auftragsbestand_aj numeric(12,2), add column if not exists auftragsbestand_nj numeric(12,2), add column if not exists wert_auslieferung numeric(12,2);
